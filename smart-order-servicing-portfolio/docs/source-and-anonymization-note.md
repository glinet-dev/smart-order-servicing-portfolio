# Source and Anonymization Note

This repository intentionally uses fictional airline names and anonymized benchmark language. The purpose is to demonstrate Product Owner thinking without implying employment, endorsement, access to internal systems, or confidential knowledge from any airline.

The airline retailing domain framing is based on publicly observable patterns such as online booking, manage booking, ancillary selling, order servicing, seller/distribution APIs, and industry movement toward Offers and Orders.

Methodology and industry references used in this public version:

- IATA Airline Retailing: https://www.iata.org/en/programs/airline-distribution/retailing/
- Scrum Guide: https://scrumguides.org/scrum-guide.html
- SAFe Lean-Agile Principles: https://framework.scaledagile.com/safe-lean-agile-principles/
- SAFe Product Owner: https://framework.scaledagile.com/product-owner/
